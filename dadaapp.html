DadaAPP

Aplicacion para Android creada en Processing. La aplicacion funciona bajo las mismas reglas de la corriente del Dadaismo, con una base de datos amplia y variada, permite crear poemas dadaistas permitiendo decidir si se quiere a base de enunciados o determinados numeros de palabras.
